Palindrome Partitioning

题目描述：LeetCode
Given a string s, partition s such that every substring of the partition is a palindrome.
Return all possible palindrome partitioning of s.
For example, given s = “aab”, return
[
[“aa”,”b”],
[“a”,”a”,”b”]
]
